루프는 시간 척도가 다르다
ACTION LOOP
행동 루프
초 ~ 분 단위
한 번의 입력과 즉각적인 피드백. 게임의 "감각"을 만든다.
예: 점프 한 번, 카드 한 장 내기, 한 발 쏘기
SESSION / MID - CORE LOOP
세션 루프
분 ~ 시간 단위
한 "판"을 끝까지 가져가는 흐름. 시작-진행-끝의 구조가 있다.
예: 한 던전 런, 한 매치, 한 챕터
META LOOP
메타 루프
일 ~ 월 ~ 시즌 단위
여러 세션에 걸친 장기 진행. 게임에 "돌아오게" 만든다.
예: 캐릭터 레벨, 시즌 패스, 컬렉션 완성, 빌드 실험
바깥쪽 루프가 안쪽 루프에 "왓 겼셸 합앳 핍늉ㅡ?"를 부여한다.

Engagement Loop ≠ Compulsion Loop
같은 메커니즘 — 다른 의도. 디자이너는 그 차이를 알아야 한다.
ENGAGEMENT LOOP
자율적 흥미
✓ 플레이어가 스스로 멈출 수 있다
✓ 보상이 예측 가능하다
✓ 진보가 스킬 기반이다
✓ 플레이를 멈춰도 잃는 게 없다
예: Hades, Hollow Knight, Tetris
COMPULSION LOOP
강박적 반복
✗ 멈추기 어렵게 설계됨 (FOMO)
✗ 보상이 변동적 · 도박적
✗ 진보가 시간 / 돈 기반
✗ 안 켜면 손실이 발생 (시들기, 패널티)
예: 공격적 가챠, P2W MMO, 강제 데일리
